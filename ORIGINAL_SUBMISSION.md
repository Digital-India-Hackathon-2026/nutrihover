## Idea
NutriHover is a decentralized, off-grid smart agriculture ecosystem. A Field Node captures real-time soil telemetry, broadcasting instantly via ESP-NOW to a remote Pump Node for offline RTC irrigation and to a Drone Node for targeted spraying. The Pump Node uses Sub-GHz LoRa to relay data kilometers away to a Home Node, which bridges to an Android app via BLE and features interactive audio notifications. NutriHover delivers autonomous precision farming with zero internet dependency.
